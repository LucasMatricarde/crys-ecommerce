**Button** — the CRYS. pill button; molten-amber `primary` for the key action, frosted-glass `secondary`, text-only `ghost`. Use when you need any tap target.

```jsx
<Button variant="primary" size="md">Adicionar</Button>
<Button variant="secondary" size="lg">Ver mais</Button>
<Button variant="ghost" size="sm">Saiba mais</Button>
<Button variant="primary" disabled>Esgotado</Button>
```

Variants: `primary` (amber fill, dark text), `secondary` (elevated surface + frost hairline), `ghost` (transparent). Sizes: `sm` / `md` / `lg`. All buttons are full-pill. Labels are set in Syne Bold; Portuguese copy by default.
