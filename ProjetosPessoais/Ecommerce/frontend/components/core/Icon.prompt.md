**Icon** — curated [Lucide](https://lucide.dev) icons (MIT), single-color, 1.75 stroke, `currentColor`. CRYS. uses a thin line-icon style throughout.

```jsx
<Icon name="cart" size={24} />
<Icon name="leaf" size={20} color="var(--accent-secondary)" />
```

Available: `cart, search, menu, user, leaf, x, chevronDown, plus, minus, star, truck, shield, heart`. Color via `color` prop or inherited `currentColor`.
